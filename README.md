### install

#### Mac

```bash
brew tap homebrew/cask-fonts
brew install --cask font-caskaydia-cove-nerd-font
```

#### Linux

```shell
git clone git@github.com:ryanoasis/nerd-fonts.git --depth=1
```

```shell
cd nerd-fonts && ./install.sh
```
