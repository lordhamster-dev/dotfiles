# dotfiles

## Mac

### Install Font

```bash
brew tap homebrew/cask-fonts
brew install --cask font-caskaydia-cove-nerd-font
```

## Archlinux

### Install Font

```bash
sudo pacman -S ttf-cascadia-code-nerd
```

